## 📋 RESUMEN - EDITAR INSTRUCTIVO (CREAR VERSIONES V2, V3, V4...)

### 🎯 ¿Qué hace?

Permite editar instructivos existentes y crear **nuevas versiones** automáticamente:

```
Instructivo #1324 Versión 1 → Editar → Guardar → Instructivo #1324 Versión 2
Instructivo #1324 Versión 2 → Editar → Guardar → Instructivo #1324 Versión 3
```

### 🧪 PRUEBA - PASOS

**1. Inicia sesión:**
```
http://localhost/instructivo/
Usuario: sarenas
Contraseña: 1234
```

**2. Ve a Editar Instructivo:**
```
http://localhost/instructivo/app/Procesos/editar_instructivo.php
```

**3. Busca un instructivo:**
- Selecciona exportadora
- Selecciona especie
- Click en "🔍 Buscar"

**4. Carga instructivo:**
- Click en "✏️ Editar / Crear Versión"
- Sistema muestra versión actual (ej: V1)
- Carga TODOS los datos (cabecera + pedidos + detalle)

**5. Edita lo que necesites:**
- Cambia fecha, turno, observación
- Agrega/elimina pedidos
- Agrega/elimina calibres con configuración

**6. Guarda nueva versión:**
- Click en "💾 Guardar como Versión 2"
- Sistema crea nueva versión automáticamente
- Mensaje: "✅ Versión 2 creada exitosamente"

### 🗄️ ESTRUCTURA DE DATOS

**Mismo ID, diferente versión:**
```
inst_cab_instructivo: id_instructivo = 1324 (mismo)
inst_pedidos: id_instructivo = 1324, version = 1 → 2 → 3
inst_detalle_instructivo: id_cab_instructivo = 1324, version = 1 → 2 → 3
```

**Cada versión es INDEPENDIENTE:**
- ✅ Misma número de instructivo
- ✅ Diferente versión
- ✅ Diferente fecha
- ✅ Diferente cantidad de pedidos/detalle
